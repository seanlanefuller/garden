# Garden Simulation

A 3D simulation of a garden built with Vite and Three.js.

## Overview

This project provides a visually rich and interactive garden environment. It explores 3D rendering techniques using Three.js and modern web development practices with Vite.

## Getting Started

To run the project locally, follow these steps:

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd garden
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`.

## Usage

- **Camera**: Use your mouse to rotate and zoom the camera.
- **Interactions**: Explore the 3D environment and interact with garden elements.
